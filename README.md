## Description

Android application designed for campus safety and communication. It enables users to send emergency alerts with live GPS location to campus security.

## Tech Stack

- Java / Kotlin
- Android Studio
- Firebase Authentication
- Firebase Firestore / Realtime Database
- Google Maps API

## Setup

Open the project in Android Studio, connect Firebase, add `google-services.json`, and run the application.
